## [1.1.0] - 2024-12-10

### Added

- Dim unmoveable pieces;
- Sound FX for time running out;
- Make numbers stand out a bit more;
- Show number of keystrokes to select a piece;

### Removed

- Removed "cycling back" of piece selection. Only cycle forward now.
