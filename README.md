# Rush Hour

Rush Hour is a block sliding game created by [Nob Yoshigahara](https://en.wikipedia.org/wiki/Nob_Yoshigahara) and published by [ThinkFun](https://www.thinkfun.com/). The game consists of a 6x6 grid with 16 vehicles of different sizes and colors. The goal of the game is to move the red car to the exit by sliding the other vehicles out of the way. The game is played by moving the vehicles horizontally or vertically along the grid. Vehicles can only move in the direction they are facing and cannot move through other vehicles. The game is won when the red car is moved to the exit.
